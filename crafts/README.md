Place race craft templates here before running `./create.sh`.

Default expected VAB craft files:

- `crafts/VAB/OA_KerbalX_Sounding.craft`
- `crafts/VAB/OA_KerbalX_Suborbital.craft`
- `crafts/VAB/OA_KerbalX_Orbiter.craft`

Optional SPH craft files can go in:

- `crafts/SPH/`

The current race simulation is still abstract, but these craft files are packaged into `Ships/` in the release zip so the content pipeline is ready for later automation work.
